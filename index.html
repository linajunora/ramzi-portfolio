BookDucks

BookDucks is a full-stack web application I developed where users can explore a collection of books, save titles to a personal “To Be Read” list, and rate the books they’ve read. Logged-in users can manage their saved books, assign ratings from 1–5, and filter their list based on author, title, or personal score.
The backend is built with Strapi, a headless CMS that allowed me to structure and manage book data, user profiles, and relationships between them. I implemented JWT authentication to protect user routes and ensure secure access. The frontend is built using vanilla JavaScript, and I used Axios to handle all communication with the backend.

Tamagotchi

Pokémon Tamagotchi is a virtual pet game where players adopt and care for their favorite Pokémon. Built with JavaScript and styled using Sass, the game uses object-oriented programming to manage each Pokémon’s stats: energy, fullness, and happiness. Users can feed, play, and let their Pokémon nap to keep them happy.
Each Pokémon has a timer that gradually reduces their stats, and all data is stored in localStorage to persist between sessions. The UI is enhanced with custom sprites fetched from the Pokémon API, and a log tracks every action taken with your Pokémon. The design was made with a retro windows 95 aesthetic in mind.

Mood Journal

Mood Journal is a reflective mood-tracking app built with JavaScript and a local JSON Server backend. Users can log how they feel on a given day by selecting a mood, choosing from a list of feelings and activities, and adding personal notes. Each entry is saved and tied to the individual user, making it possible to revisit, edit, or delete past logs. The app includes full CRUD functionality and uses sessionStorage to manage in-progress entries and edits. Styled with SCSS, the interface is designed to feel soft and calming while supporting deep emotional reflection.
